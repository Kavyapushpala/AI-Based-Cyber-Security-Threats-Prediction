## Overview

This project is a web-based dashboard that simulates and visualizes
cyber-security threats.\
It includes pages for real-time threat monitoring, ML-based threat
simulation, global attack maps, static analytics, and detailed threat
reports.

The project is built using React, Vite, TypeScript, TailwindCSS, and
ShadCN components.

------------------------------------------------------------------------

## Features

-   Real-time threat stream dashboard\
-   AI/ML simulation panel\
-   Global threat/attack map\
-   Static and dynamic data dashboards\
-   Threat reports page\
-   Responsive UI\
-   Clean and modular code structure

------------------------------------------------------------------------

## Tech Stack

-   React + Vite\
-   TypeScript\
-   TailwindCSS\
-   ShadCN UI\
-   Recharts

## How to Run Locally

1.  Clone the repository\

```{=html}
<!-- -->
```
    git clone <your-repository-link>

2.  Install dependencies\

```{=html}
<!-- -->
```
    npm install

3.  Start the development server\

```{=html}
<!-- -->
```
    npm run dev

4.  Build for production\

```{=html}
<!-- -->
```
